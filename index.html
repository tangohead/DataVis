DataVis

What is it?

DataVis is a simple Java application that displays a series of data points collected by a Lego Mindstorms robot. It was produced for use within CS313 Mobile Robotics at the University of Warwick.

How do I use it?

Getting the data off the robot

  1. Once you've run the robot on it's course, plug it into the computer using the USB cable.
  2. Find the directory where you unzipped leJOS to and browse to the bin directory.
  3. Run nxjbrowse. You can browse the robot's files here.
  4. Download data.csv and delete it from the robot.

Displaying it in DataVis

  1. In the file RobotDataVisualiser.java, change the value of the variable filename to the data you just got from the robot.
  2. Build and run!

Extra Stuff

  • Red line = raw data
  • Black line = 3 point moving average
  • Green line = 5 point moving average
  • To turn off moving averages, comment out the relevant lines in RobotDataVisualiser.java.
